## Data Directory

This directory is intended to store large data stores for gprobing.

### Ignored Stores

The following stores in this directory are ignored by git:

- `*.tar`
- `*.tar.gz`
- `*.tar.bz2`
- `*.tar.xz`
- `*.zip`
- `*.7z`
- `*.rar`
- `*.iso`
- `*.img`
- `*.bin`
- `*.dmg`
- `*.vdi`
- `*.vmdk`
- `*.vhd`
- `*.swp`
- `*.swo`
- `*.swn`
- `*.swm`
- `*.swx`
- `*.swpx`
